## 🎯 Career Goal

안드로이드 구조 설계와 운영 안정화 경험을 바탕으로,
더 큰 규모의 서비스에서 확장성과 유지보수성이 뛰어난 Android 아키텍처를 구축하는 개발자로 성장하고자 합니다.

높은 트래픽과 다양해지는 사용자 요구를 안정적으로 처리할 수 있는 서비스를 만드는 데 기여하고 싶습니다.

앞으로는 **Android만이 아니라 AI 기반 사용자 경험 개선, Hybrid·Cross-platform 기술의 연동** 등
서비스 품질과 개발 생산성을 동시에 높이는 영역까지 전문성을 넓혀가는 것이 목표입니다.
